# 科学计算器 在线工具分享

哈喽大家好，今天给大家安利一个我最近用 **Vue** 手搓的在线工具——**科学计算器**。

其实市面上计算器挺多的，手机自带的也能用。但有时候在电脑前工作学习，遇到稍微复杂点的算式，手机按着不方便，系统自带的计算器切来切去又觉得有点麻烦。

所以我就想，干脆自己做一个网页版的，打开就能用，还不用安装。

## 这个计算器有啥特别的？

### 1. 并没有那么“难”用

虽然名字叫“科学计算器”，听起来挺唬人的，但其实它的界面我尽量设计得直观清爽。
除了最基础的加减乘除，像什么平方、开根号、百分比它都在显眼的位置。
如果你是学生党或者工程师，经常要跟三角函数（sin, cos, tan）、对数（log, ln）打交道，这也不再话下。而且很贴心地做了**角度（Deg）**和**弧度（Rad）**的切换按钮，再也不怕算错数了。

### 2. 把算式显示出来

这其实是我自己的一个痛点。很多计算器是你按一个数，它显示一个数，按了“+”之后前面的数就不见了。
这个工具支持**表达式显示**，你输入的完整算式会一直停留在屏幕上。比如你算 `sin(30) + 5! / 2`，你可以清楚地看到自己是不是输对了，哪里输错了一删就能改，不用全部重来。

### 3. 记性很好的“历史记录”

有没有过那种情况：算了一大串数，结果出来后，突然忘了上一步算出来的数是多少？
我在旁边做了一个**历史记录**功能。你每一次按“=”号，它都会帮你把算式和结果存下来。点一下历史记录里的结果，它就能直接填回输入框里，让你接着算，特别方便。

而且这些记录是存在你浏览器本地的，关掉网页再打开，它还得在。

### 4. 键盘党狂喜

作为一个平时手不离键盘的人，我给它加上了全套的键盘支持。
你不需要用鼠标一个个去点按钮，直接在键盘上敲 `1+2*(3-4)` 回车，结果立马就出来了。这种行云流水的快感，谁用谁知道。
